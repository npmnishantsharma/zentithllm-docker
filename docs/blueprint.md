# **App Name**: DevCom Chat

## Core Features:

- User Authentication Flow: Provides secure login and user registration interfaces, including email/password input fields, dedicated login and sign-up buttons, and a 'Forgot password' recovery link.
- Conversations Sidebar: Displays a dynamic and searchable list of user chat conversations and contacts, featuring visual cues for active and hovered states to enhance navigation.
- Real-time Message Display: Renders messages in a clean, chronological feed, clearly distinguishing between sent and received messages, and includes subtle timestamps with smooth scrolling for a seamless experience.
- Message Input and Send: Features a fixed input bar at the bottom for composing new messages, complete with a text input area and an interactive send button supporting text or icon.
- User Presence Indicators: Integrates real-time feedback mechanisms such as a 'User is typing...' indicator and online/offline status displays for participants within a conversation.
- AI Message Context Assistant: An AI tool that analyzes the ongoing chat context to suggest intelligent, relevant replies, common commands, or code snippets, aiming to significantly enhance communication efficiency.

## Style Guidelines:

- The overall theme is dark and developer-inspired. The main background is the user-specified '#0f172a'. Secondary surfaces like panels use '#111827' and '#1f2937'. Borders are defined by '#2a2f3a', maintaining a consistent depth.
- The primary interactive color is a vibrant blue (#3b82f6), chosen for call-to-action buttons, active navigation states, and highlighting sent messages against the dark background. (HSL: H=217, S=80%, L=59%)
- A distinct accent color, purple (#856fe8), is used for non-primary highlights or specialized elements. This hue is analogous to the primary blue (H=247), providing clear visual differentiation while harmonizing with the modern, tech-focused aesthetic. (HSL: H=247, S=70%, L=65%)
- Text colors ensure readability on dark backgrounds: '#e5e7eb' serves as the primary text color, while a softer '#9ca3af' is used for secondary information such as timestamps or muted details.
- Primary UI font: 'Inter' (sans-serif), chosen for its neutrality, modern appearance, and excellent readability across the user interface, serving for headlines and body text.
- Code-style font: 'JetBrains Mono' (monospace), applied specifically for elements such as code snippets or command-line outputs, enhancing the developer-inspired aesthetic.
- Minimalist outline icons are favored, ensuring clarity and coherence with the clean, modern design. Icons should be functional and unobtrusive, avoiding heavy embellishment.
- Emphasis on clean spacing and precise alignment to prevent clutter. The login page will feature a centered, compact card for focused interaction, while the chat interface adopts a responsive layout with a dedicated left sidebar and a dynamic main chat area.
- Subtle and swift animations, with transitions typically ranging from 150ms to 200ms, provide fluid feedback on user interactions (e.g., hover effects, state changes). Active elements will feature a gentle glow for emphasis.
